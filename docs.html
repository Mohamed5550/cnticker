Customizable News Ticker (CNTicker)

Documentation for implementing and customizing the news ticker.

Introduction

The Customizable News Ticker (CNTicker) is a lightweight, easy-to-use JavaScript plugin for adding live, scrolling news or announcements to your website. It supports multiple animations, customization options, and a responsive design, making it suitable for any project.

Installation

Include the following scripts and styles in your HTML file:

                <!-- Include CNTicker -->
<script src="js/cnticker.js"></script>

Quick Start

Follow these steps to quickly set up the CNTicker:

                
<!-- HTML Code -->
<ul class="my-ticker" data-pause-on-hover="true" data-cnticker-animation="write">
    <li>Breaking: Global stocks surge amid market optimism</li>
    <li>Weather Alert: Heavy rainfall expected in the north.</li>
</ul>

<!-- JavaScript Code -->
var element = document.querySelector('.cnticker');
new CNTicker(element);

Options

You can set options in one of these two ways

new CNTicker(element, {
    speed: 40,
    animation: 'up',
    pauseOnHover: true
});

or

<ul class="my-ticker" data-cnticker-pause-on-hover="true" data-cnticker-animation="up"></ul>

Customize the ticker with the following options:

Option Type Description Default
label String Label displayed before the ticker (e.g., "Breaking News"). ''
withLabel Boolean Display or hide the label false
speed Number Scrolling speed in pixels per second. 30
pauseOnHover Boolean Determine if you want the ticker to pause on hover. false
animation String Animation type (e.g., 'marquee', 'up', 'down', 'write'). 'marquee'
dir String Text direction ('ltr' or 'rtl'). 'ltr'
transition Number Delay before moving to the next item (in other animations than 'marquee') (in ms). 5000
withSeparators Boolean Show or hide the separators between items. true
cursorSpeed Number Write character every x ms (in write animation mode). 10
fontSize Number font size in pixels. 16
labelFontSize Number Label font size in pixels. 20
labelColor String Color of the label background. '#1565c0'
borderColor String Color of the label border. '#1976d2'
background String Background of the ticker. 'linear-gradient(90deg, #64b5f6, #1976d2)'
textColor String Color of the ticker. '#FFF'
